# Password Strength Checker #

A Python-based tool to evaluate password strength using multiple security parameters.

Learning Outcome:
This project demonstrates basic cybersecurity practices like password validation and secure password design.

## Features
- Checks password strength (Weak / Medium / Strong)
- Validates:
  - Length
  - Uppercase & lowercase letters
  - Numbers
  - Special characters
- Detects common passwords (dictionary attack)
- Provides suggestions for improvement

## Tech Used
- Python
- Regular Expressions (re)

## Future Improvement
- GUI-based interface
- Integration with real-time password breach APIs
- Advanced entropy-based strength calculation
